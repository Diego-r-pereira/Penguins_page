Objetivo
General

Desarrollar un modelo predictivo en machine learning para identificar las distintas especies en peligro de extinción de pingüinos en determinadas islas del polo sur mediante los datos obtenidos de sus características físicas utilizando un método de aprendizaje automático.

Problemática

¿Cómo desarrollar un modelo predictivo machine learning de especies de pingüinos que logre generar los datos, la información de forma segura y responsable?

Metodología Crisp

Proporciona una descripción normalizada del ciclo de vida de un proyecto estándar de análisis de datos, de forma análoga a como se hace en la ingeniería del software con los modelos de ciclo de vida de desarrollo de software. El modelo CRISP-DM cubre las fases de un proyecto, sus tareas respectivas, y las relaciones entre estas tareas.

Marco teórico
Google Colab

Es un servicio gratuito de nube alojado por Google para fomentar la investigación sobre Aprendizaje de Máquina e Inteligencia Artificial. Siendo este un producto de Google Research. Permite a cualquier usuario escribir y ejecutar código arbitrario de Python en el navegador.

Random Forest Classifier

Es un método versátil de aprendizaje automático capaz de realizar tanto tareas de regresión como de clasificación. También lleva a cabo métodos de reducción dimensional, trata valores perdidos, valores atípicos y otros pasos esenciales de exploración de datos.

Streamlit

Es una biblioteca de Python de código abierto que facilita la creación de hermosas aplicaciones web personalizadas para el machine learning y la ciencia de datos.

Pandas

Es una librería de Python especializada en el manejo y análisis de estructuras de datos, y sobre todo dentro del ámbito de Data Science y Machine Learning, que ofrece unas estructuras muy poderosas y flexibles que facilitan la manipulación y tratamiento de datos.

Numpy

Es una librería de Python especializada en el cálculo numérico y el análisis de datos, especialmente para un gran volumen de datos.

Conclusiones

En función a las pruebas realizadas al modelo machine learning de predicción de especies de pingüinos, se ha arribado a las conclusiones siguientes:

Fue posible obtener el dataset con la información necesaria de las especies de pingüinos en el polo sur y así desarrollar el modelo de machine learning, como un medio facilitador tecnológico para el aprendizaje en aplicaciones de Big Data.

Con la implementación del modelo de machine learning de predicción de especies de pingüinos, se logrará mejorar la investigación y crear mejores algoritmos en el área.

Recomendaciones

En función a las pruebas realizadas al modelo machine learning de predicción de especies de pingüinos, se ha arribado a las recomendaciones siguientes:

Se recomienda el pleno uso del modelo de machine learning como medio facilitador para investigar y ayudar al problema del peligro de extinción de especies de pingüinos en el polo sur.
